This directory contains the data required to run the web server.

If you are using `hcb2py`, this directory should look like:

```
data/
  .gitignore
  __init__.py
  README.md
  iroseka.py
  irohika.py
  ...
```

If you are using `hcb2db`, this directory should look like:

```
data/
  .gitignore
  __init__.py
  README.md
  vnaas.db
```
